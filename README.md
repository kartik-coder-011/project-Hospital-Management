## 🏥 Smart Hospital Finder and Emergency Support System
🚀 **Find Nearby Hospitals Instantly with Smart Map-Based Discovery**
A modern, frontend-based healthcare web application that enables users to quickly locate nearby hospitals using an interactive map interface and intuitive design.


## 🌟 Overview
In critical situations, every second matters.
Smart Hospital Finder is designed to reduce the time and effort required to locate nearby healthcare facilities by combining **location-based visualization** with a **clean and responsive UI**.
This project demonstrates how frontend technologies can be used to build impactful real-world solutions.


## ❗ Problem Statement
In many cities, hospitals face **overcrowding issues**, especially during peak hours and emergencies.
Patients often visit hospitals without knowing availability, leading to:
* 🏥 Long waiting queues
* 😓 Overcrowded hospital environments
* ⏳ Wasted time and delayed treatment
* ❌ No clear idea of when their turn will come
Additionally, people struggle to **find the right hospital nearby** based on their needs, which further increases confusion and inefficiency.


## 💡 Our Solution
Smart Hospital Finder addresses these real-life challenges by providing a **smart and user-friendly platform** that:
* 📍 Helps users **find nearby hospitals from home** using map-based discovery
* 🕒 Introduces a **slot-based system (e.g., 10 patients per slot)** to reduce overcrowding
* ⏱️ Allows users to **wait for their turn from home** instead of standing in long queues
* 🏥 Displays structured hospital information for better decision-making
This approach not only improves patient convenience but also helps hospitals **manage crowd efficiently** and deliver better healthcare services.


## 🚀 Key Features

* 🗺️ **Interactive Map-Based Hospital Discovery**
  Users can easily locate nearby hospitals using an integrated Google Maps interface
  
* 📍 **Find Hospitals from Home**
  No need to physically search — users can explore hospitals remotely
  
* 🕒 **Smart Slot Management System**
  Hospitals can manage patients in fixed slots (e.g., 10 patients per slot) to reduce overcrowding
  
* ⏱️ **Wait from Home**
  Patients can wait for their turn comfortably instead of standing in long queues

* 🏥 **Structured Hospital Information**
  Clear display of hospital details for better decision-making

* 🔍 **Search & Navigation Friendly UI**
  Easy-to-use interface designed for quick access during emergencies

* 📱 **Fully Responsive Design**
  Works seamlessly on mobile and desktop devices

* ⚡ **Fast & Lightweight Frontend**
  Ensures smooth performance even on low-end devices

## 🚨 Emergency Assistance System
Smart Hospital Finder is designed to handle **critical emergency situations**, where quick response and proper guidance can save lives.

### ⚡ Emergency Features

* 🚑 **Ambulance Call Support**
  Users can quickly request an ambulance during emergencies

* 📍 **Live Location Sharing**
  Allows users to share real-time location for faster and accurate response

* 🗺️ **Nearby Hospitals Display**
  Instantly shows nearby hospitals on the map for quick decision-making

* 🚑 **Ambulance Tracking (Demo Feature)**
  Users can track ambulance movement (currently in demo version)

* 📞 **Emergency Contact Option**
  Quick access to emergency contact numbers

* ☎️ **Essential Helpline Numbers**
  Provides important healthcare and emergency helplines

* 🕒 **Smart Slot Management**
  Reduces overcrowding by distributing patients into fixed slots


### 🧭 Emergency User Guidelines
To help users act effectively during emergencies, the system also provides:
* 📌 Step-by-step guidance on what to do in emergency situations
* 🏠 Advice to stay calm and avoid panic
* 📍 Instructions to share correct location details
* 🚑 Guidance on when to call an ambulance
* 🏥 Suggestions for selecting the nearest suitable hospital

* 📍 Improves hospital accessibility
* 🚑 Enhances emergency response coordination
* 🧠 Helps users take informed decisions under stress

## ⚙️ Getting Started

### 🔧 Run Locally

```bash
# Clone repository
git clone https://github.com/your-username/smart-hospital-finder.git

# Open project folder
cd smart-hospital-finder

# Run
Open index.html in your browser
```

## 🔐 Google Maps Configuration
1. Visit Google Cloud Console
2. Enable **Google Maps JavaScript API**
3. Generate your API key
4. Replace it in your JavaScript file
